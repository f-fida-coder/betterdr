Baccarat – Documentation

Introduction

Baccarat is a responsive HTML5 casino game built using Phaser 3. It is optimized for desktop and mobile browsers and designed for Codecanyon distribution.

Features

  • Phaser 3 based
  • Responsive layout
  • Optimized loading architecture
  • Fullscreen support
  • Local gameplay mode

Scene Flow

BootScene → SplashScene → MenuScene → LocalScene

How to Run

Use a local web server (Live Server, XAMPP) or upload to Netlify. Do not open index.html directly.

Customization

Replace assets inside the assets/ folder while keeping filenames unchanged.

Fullscreen Support

Fullscreen can be toggled using the fullscreen icon. Layout and animations automatically adjust on resize.
