# Admin Panel - Quick Start Guide

## How to Access the Admin Panel

### Step 1: Log In
1. Open the application
2. Register or log in with existing credentials
3. You'll be directed to the main dashboard

### Step 2: Click Admin Panel Button
- Look for the **"Admin Panel"** button in the top-right corner of your dashboard
- Click it to enter the admin interface

### Step 3: Navigate the Dashboard
The main screen shows a grid of colored cards representing different admin functions.
